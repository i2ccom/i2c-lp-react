# Service Visual Concepts (Glassy Vector 3D)

## Goal
Upgrade service visuals to look:
- more product-specific (not generic)
- glassy corporate style
- vector-like 3D abstraction
- business-friendly and premium

## Visual Direction
- Style: Glassmorphism + vector 3D scene
- Lighting: Soft cyan/teal highlights on deep navy-neutral backgrounds
- Composition: Clean central product concept with 2-4 supporting UI modules
- Tone: Executive-ready, enterprise SaaS, modern Asian-market friendly
- Exclusions: no text overlays, no watermarks, no logos embedded in image

## Use in Pages
- `/services` hero: use `corporate-users-asia.jpeg` directly below heading/subtitle.
- `/services/:slug` detail: show two product UI previews plus one **product concept image** (service-specific, replacing generic corporate image in detail page section).
